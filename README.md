# RadioPartsDB

Приложение для учёта радиокомпонентов и быстрого поиска по местам хранения.

---

## ✨ Возможности

### Основные
- 📦 **Полный CRUD** – добавление, редактирование, удаление деталей.
- 📂 **Дерево категорий** – иерархическая структура (~250 категорий), рекурсивная фильтрация (показывает детали из подкатегорий).
- 📍 **Навигатор по местам хранения** – адресация вида `Место / Контейнер / Полка / Секция`, клик по ветке фильтрует таблицу.
- 🎨 **Цветовая индикация таблицы** – фон строки зависит от количества: 🟢 зелёный (>10), 🟡 жёлтый (1-9), 🔴 красный (0). Ячейка «Состояние» имеет собственный цвет (Новое – зелёный, Б/У – голубой, Неисправно – красный и т.д.).
- 📷 **Предпросмотр фото и даташитов** – поддержка локальных файлов и URL (из интернета).
- 🔍 **Быстрые фильтры** – «Все», «В наличии», «Мало», «Нет».
- 📥 **Импорт из CSV** – автоопределение кодировки, очистка HTML, парсинг дат.
- 💾 **Автоматическое сохранение состояния** – размеры окна, ширина колонок, порядок колонок, глубина деревьев.

### Новое в версии 0.21.0 (и выше)

#### 👁️ Окно просмотра детали
- **Двойной клик** по строке таблицы открывает немодальное окно «Просмотр детали» (только чтение).
- Отображаются **все поля** детали, включая размеры конденсаторов, номинал с единицей измерения, заметки.
- Изображение (локальное или из интернета) масштабируется под размер окна; кнопка **«Увеличить»** открывает отдельное окно с масштабированием **колёсиком мыши** и перетаскиванием.
- Кнопки:
  - **«Редактировать»** – открывает диалог редактирования (после сохранения данные обновляются и в окне просмотра, и в главной таблице).
  - **«Печать / PDF»** – печать карточки детали или сохранение в PDF.
  - **«Сохранить фото»** – сохранить изображение на диск.

#### ✏️ Пакетное редактирование
- Выделите несколько деталей (зажимая `Ctrl` или `Shift`), нажмите кнопку **«Пакетное редактирование»**.
- Массово измените:
  - **Категорию** (выбор из дерева)
  - **Место хранения**
  - **Статус**
  - **Тип детали** (с добавлением нового в словарь)
  - **Производителя** (с добавлением нового в словарь)
  - **Цену** (установить, увеличить, уменьшить)
  - **Количество** (установить, добавить, отнять)
- Все изменения применяются одним нажатием.

#### ⌨️ Горячие клавиши
| Комбинация | Действие |
|------------|----------|
| `Ctrl+N`   | Новая деталь |
| `Ctrl+E`   | Редактировать выбранную деталь |
| `Del`      | Удалить выбранную деталь (с подтверждением) |
| `Ctrl+F`   | Переместить фокус в поле поиска |
| `F5`       | Обновить все данные (таблицу, категории, навигатор) |

#### 🧩 Умная сборка названия
- В диалоге редактирования заполните **номинал** (число) и выберите **единицу измерения** (Ом, кОм, МОм, мкФ, нФ, пФ и т.д.).
- Нажмите кнопку **«Собрать название»**.
- Программа автоматически сформирует имя в формате:  
  `[номинал][единица] [напряжение/мощность] [корпус]`  
  (например, `10мкФ 25В 10x17` для конденсатора или `100кОм 0.5Вт 0805` для резистора).
- Напряжение (например, `25V`) извлекается из последнего сегмента категории и преобразуется в `25В`.
- Мощность (например, `0.5 W`) также извлекается из категории.

#### 📏 Размеры конденсаторов
- При выборе категории, содержащей слово **«Конденсатор»**, в диалоге редактирования появляются поля:
  - Диаметр (мм)
  - Высота (мм)
  - Шаг выводов (мм)
  - Толщина выводов (мм)
- Введённые размеры отображаются в таблице в колонке **«Корпус»** в формате `⌀10×17мм`, `шаг 5мм`, `вывод 0.5мм`.

#### 📋 Выпадающие списки для типа детали и производителя
- Поля **«Тип детали»** и **«Производитель»** теперь являются выпадающими списками с возможностью **ввода новых значений**.
- Новые значения автоматически сохраняются в базу данных и появляются в списке при следующем вводе.
- При первом запуске словари заполняются начальными значениями (например, «Резистор», «Конденсатор», «Texas Instruments» и т.д.).

#### 📷 Индикатор фото в таблице
- В колонке **«Корпус»** перед текстом отображается 📷, если у детали указан путь к изображению.

#### 📅 Кнопка «Сегодня» для даты ревизии
- В диалоге редактирования рядом с полем «Дата ревизии» появилась кнопка **«📅 Сегодня»**, устанавливающая текущую дату.

#### 💾 Улучшенное сохранение состояния окна
- Программа запоминает не только размер и позицию окна, но и **развёрнутое состояние** (максимизировано / обычное).
- После перезапуска окно восстанавливается в том же виде, в котором было закрыто.

#### 🌐 Русифицированное контекстное меню
- Во всех текстовых полях, полях ввода комбобоксов, заметках и числовых полях контекстное меню полностью переведено на русский язык:
  - Отменить, Повторить, Вырезать, Копировать, Вставить, Удалить, Выделить всё.

## 🖥️ Desktop (PySide6)

- **Тёмная тема** (в планах).
- **Экспорт в CSV/Excel** (в планах).
- **Уведомления о низких остатках** (в планах).
- **Расширенный поиск** (в планах).

---

## 🌐 Веб-интерфейс (SPA)

- **Динамическая загрузка** категорий и деталей (без перезагрузки страницы).
- **Фильтрация** по категориям, наличию.
- **Живой поиск** по таблице.
- **Адаптивный дизайн** (Bootstrap 5).
- **Цветовая схема**, аналогичная Desktop (фон строки по количеству, цвет статуса).

> **Примечание:** веб-клиент находится в стадии эксперимента (ветка `spa-experiment`). Для полноценного использования рекомендуется Desktop-версия.
