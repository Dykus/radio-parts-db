# RadioPartsDB

Приложение для учёта радиокомпонентов и быстрого поиска по местам хранения.

---

## ✨ Возможности

### Основные
- 📦 **Полный CRUD** – добавление, редактирование, удаление деталей.
- 📂 **Дерево категорий** – иерархическая структура (~250 категорий), рекурсивная фильтрация (показывает детали из подкатегорий).
- 📍 **Навигатор по местам хранения** – адресация вида `Место / Контейнер / Полка / Секция`, клик по ветке фильтрует таблицу.
- 🎨 **Цветовая индикация таблицы** – фон строки зависит от количества: 🟢 зелёный (>10), 🟡 жёлтый (1-9), 🔴 красный (0). Ячейка «Состояние» имеет собственный цвет (Новое – зелёный, Б/У – голубой, Неисправно – красный и т.д.).
- 📷 **Предпросмотр фото и даташитов** – поддержка локальных файлов и URL (из интернета).
- 🔍 **Быстрые фильтры** – «Все», «В наличии», «Мало», «Нет».
- 📥 **Импорт из CSV** – автоопределение кодировки, очистка HTML, парсинг дат.
- 💾 **Автоматическое сохранение состояния** – размеры окна, ширина колонок, порядок колонок, глубина деревьев.

### Новое в версии 0.22.0

#### ☁️ Резервное копирование в Яндекс.Диск
- Кнопки в меню «Файл»: «Выгрузить резервную копию» и «Восстановить из резервной копии».
- Архив включает базу данных `radioparts.db`, папки `images` и `datasheets`.
- Отложенное восстановление: программа скачивает архив, закрывается и при следующем запуске восстанавливает данные.

#### 📤 Экспорт в CSV и Excel
- Экспорт текущей (отфильтрованной) таблицы.
- CSV сохраняется в UTF-8 with BOM (корректная работа с кириллицей в Excel).
- Excel-файл имеет автоматически подогнанную ширину колонок.

#### ✏️ Пакетное редактирование
- Выделите несколько деталей (зажимая `Ctrl` или `Shift`), нажмите кнопку **«Пакетное редактирование»**.
- Массовое изменение категории, места, статуса, типа детали, производителя, цены, количества.

#### 👁️ Окно просмотра детали
- **Двойной клик** по строке таблицы открывает немодальное окно «Просмотр детали».
- Отображаются все поля, изображение (локальное/URL), даташит.
- Кнопка **«Увеличить»** открывает отдельное окно с масштабированием колёсиком мыши.

#### ⌨️ Горячие клавиши
| Комбинация | Действие |
|------------|----------|
| `Ctrl+N`   | Новая деталь |
| `Ctrl+E`   | Редактировать выбранную деталь |
| `Del`      | Удалить выбранную деталь (с подтверждением) |
| `Ctrl+F`   | Переместить фокус в поле поиска |
| `F5`       | Обновить все данные |

#### 🧩 Умная сборка названия
- В диалоге редактирования введите **число** и выберите **единицу измерения** (Ом, кОм, МОм, мкФ, нФ, пФ и т.д.).
- Нажмите **«Собрать название»** – программа автоматически сформирует имя в формате `[номинал][единица] [напряжение/мощность] [корпус]`.

#### 📏 Размеры конденсаторов
- При выборе категории, содержащей слово **«Конденсатор»**, появляются поля: диаметр, высота, шаг выводов, толщина выводов.
- Размеры отображаются в таблице в колонке «Корпус» в формате `⌀10×17мм`, `шаг 5мм`, `вывод 0.5мм`.

#### 📋 Выпадающие списки для типа детали и производителя
- Поля **«Тип детали»** и **«Производитель»** – выпадающие списки с возможностью **ввода новых значений**.
- Новые значения автоматически сохраняются в базу данных.

#### 📷 Индикатор фото в таблице
- В колонке **«Корпус»** перед текстом отображается 📷, если у детали есть изображение.

#### 📅 Кнопка «Сегодня» для даты ревизии
- В диалоге редактирования рядом с полем «Дата ревизии» появилась кнопка **«📅 Сегодня»**.

#### 💾 Улучшенное сохранение состояния окна
- Запоминается размер, позиция и развёрнутое состояние.

#### 🌐 Русифицированное контекстное меню
- Во всех текстовых полях и заметках контекстное меню полностью переведено на русский язык.

## 🖥️ Desktop (PySide6)

- Все основные функции реализованы и стабильны.
- Поддерживаются экспорт, резервное копирование, горячие клавиши, умная сборка названия, размеры конденсаторов.

---

## 🌐 Веб-клиент (SPA)

- **Динамическая загрузка** категорий и деталей (без перезагрузки страницы).
- **Фильтрация** по категориям, наличию, текстовый поиск.
- **Цветовая схема**, аналогичная Desktop (фон строки по количеству, цвет статуса).
- **Модальное окно редактирования** со всеми полями, включая номинал, единицы измерения, размеры конденсаторов.
- **Полная синхронизация с Desktop** (отображаются номинал, размеры, иконка фото).
- Работает с бэкендом FastAPI (порт 8000).

## 🚀 Установка и запуск

### Требования
- Python 3.10 или выше
- Git (опционально)

### Быстрый старт (Windows)
1. Клонируйте репозиторий (или скачайте ZIP):
   ```bash
   git clone https://github.com/Dykus/radio-parts-db.git
   cd radio-parts-db
Запустите run.bat – он автоматически создаст виртуальное окружение, установит зависимости, запустит Backend API, Desktop-приложение и веб-сервер.

Откроется браузер с веб-клиентом (http://localhost:8080).

Запуск вручную
bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск Desktop-приложения
python main.py

# Запуск Backend API (для веб-клиента)
python backend/main.py

# Запуск веб-сервера для фронтенда (из папки frontend)
python -m http.server 8080
