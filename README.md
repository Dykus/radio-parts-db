# RadioPartsDB
Приложение для учёта радиокомпонентов и быстрого поиска по местам хранения

✨ Возможности
✅  Быстрый поиск по месту хранения  — навигатор по ящикам, полкам, шкафам 
✅  Импорт из CSV  — поддержка экспорта из Memento Database, Excel 
✅  Предпросмотр фото  — загрузка изображений из интернета и локальных файлов 
✅  Дерево категорий  — удобная навигация по типам компонентов 
✅  Цветовая индикация  — зелёный (много), жёлтый (мало), красный (нет) 
✅  Архивация  — удалённые компоненты сохраняются для восстановления 
✅  Кроссплатформенность  — работает на Windows, Linux, macOS

Установка
Установите Python 3.10+
Клонируйте репозиторий:
git clone https://github.com/Dykus/radio-parts-db.git
cd radio-parts-db
