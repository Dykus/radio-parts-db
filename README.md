# 📦 RadioPartsDB
Приложение для учёта радиокомпонентов. Открытый исходный код, кроссплатформенность, синхронизация через облако.

> ⚠️ Статус: В разработке (v0.1.0)

## 🚀 Быстрый старт
1. Установите Python 3.10+
2. python -m venv venv
3. venv\Scripts\activate (Win) или source venv/bin/activate (Linux/macOS)
4. pip install -r requirements.txt
5. python main.py

## ☁️ Синхронизация
Поместите папку data/ в облако и запускайте с переменной:
RADIOPARTS_DATA_DIR=/путь/к/облаку/RadioPartsDB python main.py
