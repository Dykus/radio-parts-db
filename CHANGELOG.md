# Журнал изменений (Changelog)

Все изменения в проекте `radio-parts-db` будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
версионирование — на [Semantic Versioning](https://semver.org/lang/ru/).

---

## [0.5.0] — 2026-05-20

### 🚀 Добавлено
- **Навигатор по местам хранения**
  - Визуальное дерево всех мест хранения из базы данных
  - Автоматическая группировка по структуре (Дом → Подвал → Ящик)
  - Фильтрация таблицы при клике на место
  - Интеграция в правую панель вместе с предпросмотром фото

### ✨ Изменено
- Улучшен интерфейс: правая панель теперь разделена на фото и навигатор
- Статус-бар теперь показывает текущий фильтр по месту
- Версия приложения обновлена до 0.5.0

---

## [0.4.0] — 2026-05-20

### 🚀 Добавлено
- **Быстрые фильтры наличия**
  - Кнопки: Все / В наличии / Мало (<10) / Нет на складе
  - Мгновенная фильтрация таблицы по статусу наличия
  - Визуальное выделение активного фильтра

- **Панель предпросмотра изображений**
  - Отображение фото компонента при клике на строку
  - Поддержка локальных файлов и URL из интернета
  - Обработка нескольких ссылок через запятую
  - Загрузка изображений с сайтов (Memento, Google, ChipDip)
  - Отображение краткой информации о компоненте под фото

### 🐛 Исправлено
- Ошибка импорта типа `QDate` (перенесён из QtGui в QtCore)
- Исправлено заполнение даты ревизии в форме редактирования
- Контрастность текста в таблице (чёрный текст на цветном фоне)

---

## [0.3.0] — 2026-05-20

###  Добавлено
- **Дерево категорий слева**
  - Иерархическая навигация по категориям компонентов
  - Фильтрация таблицы при клике на папку

### ✨ Изменено
- Улучшена структура базы данных для поддержки иерархии категорий
