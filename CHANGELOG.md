# Журнал изменений (Changelog)

Все изменения в проекте `radio-parts-db` будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
версионирование — на [Semantic Versioning](https://semver.org/lang/ru/).

---

## [0.7.0] — 2026-05-21

### 🚀 Добавлено
- **Четырехуровневая структура места хранения**
  - Добавлено четвертое поле "Секция/№" в форму добавления и редактирования компонентов
  - Поддержка сложных адресов хранения, например: `Дом / Подвал / Стеллаж А / Полка 2`
  - Каскадное заполнение выпадающих списков: выбор предыдущего уровня фильтрует варианты следующего

### 🔧 Улучшено
- **Логика навигатора по местам**
  - Дерево мест теперь корректно отображает и фильтрует данные с глубиной вложения до 4 уровней
  - Автоматическое раскрытие веток при поиске детали в таблице работает для любой глубины пути
  - Сохранение и восстановление позиции окна продолжает работать стабильно

---

## [0.6.2] — 2026-05-21

### 🎨 Улучшено
- **Адаптивный светлый интерфейс**
  - Исправлены цвета правой панели (предпросмотр фото, навигатор по местам) для корректного отображения в стандартной светлой теме Windows
  - Текст стал темно-серым (`#333`, `#555`) для обеспечения высокой контрастности и читаемости
  - Фон элементов заменен с черного (`#1e1e1e`) на нейтральный светло-серый (`#f0f0f0`)
  - Дерево навигатора теперь использует стандартное синее выделение Windows

### 💾 Добавлено
- **Сохранение положения окна**
  - Программа теперь запоминает последнюю позицию и размер окна при закрытии
  - При следующем запуске окно открывается в том же месте, где было закрыто
  - Настройки сохраняются в локальный файл `window_settings.json`

---

## [0.6.1] — 2026-05-21

### 🐛 Исправлено
- **Обновление дерева мест при редактировании**
  - Исправлена ошибка, из-за которой новые места хранения (например, "Гараж") не появлялись в навигаторе после сохранения изменений в форме редактирования
  - Добавлен автоматический вызов `_load_locations()` в методах `_edit_part` и `_add_part`
  - Дерево мест теперь всегда актуально и отражает реальные данные из базы

---

## [0.6.0] — 2026-05-21

### 🚀 Добавлено
- **Двусторонняя синхронизация (Таблица ↔ Навигатор)**
  - При клике на строку в таблице, в дереве мест справа автоматически выделяется нужный элемент
  - Дерево автоматически раскрывает все ветки до нужной ячейки
  - Автоматическая прокрутка к выделенному элементу в дереве
  - Сброс выделения в дереве при клике на пустое место или деталь без места

### 🔧 Улучшено
- **Логика импорта CSV**
  - Корректная сборка пути из трёх полей: `Место` / `Контейнер` / `Ячейка`
  - Исправлено создание структуры дерева для вложенных элементов
  - Поддержка объединения дублирующихся заголовков в CSV
- **Обработка изображений**
  - Поддержка отображения локальных файлов и ссылок из интернета
  - Обработка множественных ссылок в одном поле
